# fluent-plugin-werkzeug-profiler

## Component

### WerkzeugProfilerInput

## Installation

```
$ gem install fluent-plugin-werkzeug-profiler
```

## Configuration

### WerkzeugProfilerInput

```
<source>
  type werkzeug
  path path/to/werkzeug.log
  tag werkzeug.webserver
  time_format %d-%b-%Y:%H:%M:%S
</source>
```
